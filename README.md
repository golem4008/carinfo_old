# 车企销量数据分析平台

一个基于 React 和 TypeScript 的车企销量数据分析平台，用于全面分析各车企销量表现、市场份额和增长趋势。

## 技术栈

- React 18+
- TypeScript
- Tailwind CSS
- Recharts (数据可视化)
- Framer Motion (动画效果)
- Vite (构建工具)

## 快速开始

### 安装依赖

## 如何使用

### 1. 数据文件准备

应用通过读取静态 CSV 文件获取销量数据。您需要按照指定格式准备数据文件：

1. 文件位置：`src/mocks/car_sales_data.csv`
2. 文件格式：CSV 格式，以月份为单位组织数据

### 2. CSV 数据格式要求

CSV 文件应按照以下格式组织数据：

```
YYYY年MM月数据
厂商名称,品牌名称,车型名称,车辆类型,售价,销量
特斯拉,特斯拉,Model 3,轿车,23.19,8000
特斯拉,特斯拉,Model Y,SUV,26.39,4000
...
```

**格式说明**：
- 每个月份数据前需要有月份标识行，格式为 `YYYY年MM月数据`
- 每个月份数据包含以下字段：
  - 厂商名称：如"特斯拉"、"比亚迪"等
  - 品牌名称：如"特斯拉"、"比亚迪王朝系列"等
  - 车型名称：具体车型型号，如"Model 3"、"汉EV"等
  - 车辆类型：如"轿车"、"SUV"、"小型车"等
  - 售价：车辆售价，数值类型
  - 销量：车辆销量，整数类型

### 3. 更新数据文件

1. 打开 `src/mocks/car_sales_data.csv` 文件
2. 按照上述格式修改或替换数据内容
3. 保存文件后重新启动应用，新数据将自动加载

## 功能介绍

### 数据概览
- 年度总销量展示
- 月度平均销量统计
- 同比增长率分析

### 图表分析
- 月度销量对比图表
- 市场份额分布图表
- 销量趋势分析图表
- 同比环比增长率分析图表

### 数据筛选
- 通过日期筛选器选择分析时间段
- 支持选择不同车企进行数据比较
- 表格数据支持排序和筛选功能

### 其他功能
- 支持明暗主题切换
- 响应式布局，适配不同设备

## 注意事项

1. 确保 CSV 文件格式正确，特别是月份标识行和字段顺序
2. 如需修改数据，直接编辑 `src/mocks/car_sales_data.csv` 文件即可
3. 应用会在启动时自动解析并加载 CSV 文件中的数据
4. 如果数据格式错误，应用可能无法正确展示分析结果
5. 所有数据仅在本地浏览器中处理，不会上传到服务器

## 开发说明

如需扩展功能或修改代码，请参考项目结构组织您的开发工作：
- `src/components/`: 包含所有 UI 组件

- `src/mocks/`: 包含模拟数据和 CSV 文件
